# Pensioners Distribution Dashboard

## Overview
This interactive dashboard provides a dynamic way to explore the Pensioners Distribution Data for 2022. Users can analyze the distribution of pensioners by gender and quarter and customize their views to meet specific needs.

## Features
- View data by gender and quarter.
- Customize data representations using filters.
- Download visualizations for reports.
- Includes integrated case studies for practical applications.

## Prerequisites
- Install Python 3.x.
- Install required libraries: `dash`, `pandas`, `plotly` using `pip install dash pandas plotly`.
- Download the dataset: `Distribution_of_Pensioners_2022.xlsx`.

## Usage
1. Place the dataset in the same directory as the script.
2. Run the script using `python app.py`.
3. Open the dashboard in your browser (default: `http://127.0.0.1:8050`).
4. Use the dropdown filters to interact with the data.

## Customization
- Modify the dataset to include additional metrics if needed.
- Extend the layout to include more visualizations.
